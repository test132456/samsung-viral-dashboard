# 자료실 (reference)

심의전 원고 검수가 **자동으로 사용하는 최신 약관·작성가이드**를 이 폴더에 둡니다.
해외여행보험 바이럴은 약관·가이드가 고정이라, 여기 한 번만 올려두면 팀원들이 매번 업로드할 필요가 없습니다.

## 파일 규칙
- 약관: `약관*.pdf` (또는 `.docx` / `.txt`)
- 작성 가이드: `*가이드*.pptx` (또는 임의의 `.pptx`)

## 최신화 방법
파일을 새 버전으로 교체하고 커밋 → 재배포하면 모두 최신본을 사용합니다.
(개별 검수 시 "이번 검수만 다른 파일 쓰기"로 일회성 교체도 가능)
